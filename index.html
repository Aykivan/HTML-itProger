Атрибуты

Атрибут - дополнительный функционал какого-либо тега.
Бывают атрибуты, которые подходят для всех тегов, и те, который подходят для определенных тегов.
У одного тега может быть несколько атрибутов.

  • title - всплывающая подсказка
  • width - ширина
  • height - высота

Теги для подключения JS и CSS

Тег link - прописывается в шапке. Мы добавляем туда ряд атрибутов:

  • rel="stylesheet" - мы говорим о том, что подключаем таблицу стилей
  • href="путь к CSS файлу"

Тег script - прописываем в конце, перед концом тела - body. Для того, что бы указать адресс для JS нужно спользовать атрибут - src

Теги div и span

Просто текст, как в теге p, но без доп. отступов. Он необходим для создания блока, к которому протом с помощью CSS мы будем добавлять различные стили.

span - это строчный тег. Он не заключает блоки в абзатцы. Его исполбзуем, чтобы выделить конкретный текст в определенных блоках.

Новые теги и атрибуты:

  • meta name="author" content="Айкин Иван" - автор и его имя
  • meta name="description" - описание страницы
  • meta name="keywords" - ключевые слова для поисковых систем
  • abbr title="Расшифровка абривиатуры", title можно довабивить к любому тегу, как доп описание
  • href="mailto: - позволяет создавать ссылки на почту
