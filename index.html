Атрибуты

Атрибут - дополнительный функционал какого-либо тега.
Бывают атрибуты, которые подходят для всех тегов, и те, который подходят для определенных тегов.
У одного тега может быть несколько атрибутов.

  • title - всплывающая подсказка
  • width - ширина
  • height - высота

Теги для подключения JS и CSS

Тег link - прописывается в шапке. Мы добавляем туда ряд атрибутов:

  • rel="stylesheet" - мы говорим о том, что подключаем таблицу стилей
  • href="путь к CSS файлу"

Тег script - прописываем в конце, перед концом тела - body. Для того, что бы указать адресс для JS нужно спользовать атрибут - src

Теги div и span

Просто текст, как в теге p, но без доп. отступов. Он необходим для создания блока, к которому протом с помощью CSS мы будем добавлять различные стили.

span - это строчный тег. Он не заключает блоки в абзатцы. Его исполбзуем, чтобы выделить конкретный текст в определенных блоках.

Формы

Тег - form, используется для создания Форм


Атрибут action нужен для указания сайта, на который будут отправлятся все введеные данные
Атрибут metod - get или post - сообщает о методе отправки
  • get - передача через url (адресные строки)
  • post - метод скрытой передачи данных
Тег label - выводит на экран текст
Тег input - одиночный, принимает информацию введенную пользователем. Его атрибуты:
  • placeholder - подсказка ввода в форме
  • value - значение, которое будет стоять по умолчанию в форме
  • maxlength - ограничивает длину вводимого текста
  • hidden - скрывает поле ввода
  • disabled - поле будет не активно
  • readonle - позволяет только считывать данные в форме
  • name - задаем щначение для JS
  • id - ункальный идентификатор, в основном для CSS стилей
  • for - тег для атрибута label, указывает с каким imput ему нужно работать
Фораматы type у input полей:
  • type="password" - заполнение формы будет происходить звездочками
  • type="email" - форма будет ожидать коррекнтное написание адресса почты
  • type="url" - аналогично майл
  • type="color" - позволяет сделать форму с выбором цвета
  • type="date" - создается календарь
  • type="time" - выбор времени
  • type="checkbox" - поле с галочкой
  • type="radio" - позволяет выбрать крудочком, например пол м/ж
  • type="file" - позволяет загрудать файлы
  • type="button" - кнопка без перезагрузки страницы. submit - кнопка отправки данных с перезагруской страницы
  • type="reset" - позволяет очистить форму до стартовых значений

Поля ввода текста. Тег кнопки.

Если мы хотим создать кнопку вне формы, лучше использовать - button - парный тег.

Селектор выбора

Необходимые знания:

Селектор выбора задается с помощью парного тега - select. Что бы добавить в него варианты выбора необходимо использовать парный тег - option.
Атрибуты:

  • selected - введя его выберете базовое значение
  • disabled - делает на активным один из пунктов
  • multiple - делает select многовыборным, в таком случаем модно несколько раз использовать selected

Специальные HTML5 теги

Тег - header - парный тег шапки сайта.
Тег - main - парный тег для основной части сайта.
Тег - aside - парный тег для боковой части сайта, может быть внутри других тегов, типа main.
Тег - footer - парный тег для создания футора сайта.

Шкала прогресса, заполняется с помощью атрибутов:

  • max - максимальное значения в прогрессе, в нашем случае 100
  • value - текущее значение прогресса, у нас 50

Подробнее Тег - details - парный тег для создания вкладки с доп. информацией.
Указав в нем тег - summary - мы также выбираем в виде какого слова он будет у нас отображаться.

Не все теги поддерживаются браузерами. Для корректной их работы неоюходимо их тестировать во всех популярных браузерах:

  • Google
  • Opera
  • Safari
  • Edge
  • Explorer
  • FireFox
Существуют специальные сервисы для тестирования сайта, необходимо ввести в поисковике: Тестирование в браузерах.

Новые теги и атрибуты:

  • meta name="author" content="Айкин Иван" - автор и его имя
  • meta name="description" - описание страницы
  • meta name="keywords" - ключевые слова для поисковых систем
  • abbr title="Расшифровка абривиатуры", title можно довабивить к любому тегу, как доп описание
  • href="mailto: - позволяет создавать ссылки на почту
